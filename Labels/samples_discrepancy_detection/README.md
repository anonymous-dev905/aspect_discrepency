## ner_labels.csv
| Schema | Description |
|-|-|
| Database | Name of the database  |
| Id | Id of the database report, can be used to reach original website |
| CVE | Corresponding CVE Id of the report |
| Contents | Contents of the report (may take up multiple rows) |
| Product Name | Vulnerable product name |
| Version | Vulnerable product version |
| Vulnerability Type | Vulnerability Type |
| Vulnerable Component | Vulnerable product component |
